Protocol Repository

July 12, 2019

Release 0.1 for the protocol repository

In addition to the Quantum Protocol Zoo hosted at https://wiki.veriqloud.fr we have started implementing some of the protocols using simulaQron and CQC. This repository is hosted at github https://github.com/quantumprotocolzoo/protocols.

It already contains a few protocols. More will be added in the future. We also plan to increase the robustness of these implementations and their functionality, ultimately giving access to a protocol library to be used in more complex situations.
