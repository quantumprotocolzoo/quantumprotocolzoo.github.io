Weak String Erasure

For this v0.1.2 release, we implemented the Weak String Protocol described at https://wiki.veriqloud.fr/index.php?title=Weak_String_Erasure. It is based on Alice sending qubits encoded in BB84 states and Bob measuring them at the other end. When Bob and Alice share the same basis they are garanteed to get the same results, yet if Alice is giving her preparation basis to Bob, but Bob does not give the measurement basis to Alice she cannot know which bits they have in common.

Protocol Repository

Release 0.1 for the protocol repository In addition to the Quantum Protocol Zoo hosted at https://wiki.veriqloud.fr we have started implementing some of the protocols using simulaQron and CQC. This repository is hosted at github https://github.com/quantumprotocolzoo/protocols. It already contains a few protocols. More will be added in the future. We also plan to increase the robustness of these implementations and their functionality, ultimately giving access to a protocol library to be used in more complex situations.

Quantum Protocol Zoo's Repository

What will you find here? The Quantum Protocol Zoo aims at providing an oerview of selected quantum communication protocols arranged into functionalities, protocols per se and resources needed to implement them. This repo is tightly integrated with the zoo as it contains simulaQron code for implementing these protocols using quantum optics devices or simulators.
